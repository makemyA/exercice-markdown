# Les insertions en Markdown

## Table du contenu

1. Images
2. Vidéos
3. Liens ?

## Images

Pour insérer des images, il faut utiliser la syntaxe suivante:

    ![Description de l'image](URL de l'image)

Exemple:

![Flies having a good time](https://i.redd.it/xpx53459sbw01.jpg)

## Vidéos

Il n'est pas possible d'implanter une vidéo avec Markdown, mais il est suggéré d'afficher une miniature cliquable qui amènera vers la vidéo désirée. Sous forme de code:

    [![Description de l'image](http://img.youtube.com/vi/YOUTUBE_VIDEO_ID_HERE/0.jpg)](http://www.youtube.com/watch?v=YOUTUBE_VIDEO_ID_HERE)

Exemple:

[![What is Love](http://img.youtube.com/vi/HEXWRTEbj1I/0.jpg)](http://www.youtube.com/watch?v=HEXWRTEbj1I)
